# Vindexa Website

Marketing site for [Vindexa](https://vindexa.org) — an AI study assistant that turns your course materials into quizzes, practice problems, and notes.

The main app lives in a separate repo. This one is the public-facing landing page, built in plain HTML/CSS so it stays fast and easy to iterate on.

## About Vindexa

Vindexa ingests the files from a course (lectures, slides, readings), chunks and embeds them, and retrieves the relevant context at query time. Instead of generic AI output, you get quizzes, practice problems, and condensed notes that are actually grounded in your own syllabus.

Built with Next.js, Supabase, and the OpenAI API.
